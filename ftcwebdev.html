view Code on github

Project Background

The topic of this website is the Front End Web Developer Course at Finglas Training Centre. The client is Finglas Training Centre. The target audience is is anybody looking to study front end web development in dublin and People looking to find out more about front end web development

Planning & Design

During the research phase, I researched other web developer course, college and education training websites for inspiration. The colour scheme throughout the website is quite minimal. I wanted to keep it to one or two main colours with a hint of some brighter colours throughout. I choose to use navy blue as it is professional and clean against the white background.

Development Process

The website was developed using the latest versions of HTML5, CSS3 and JavaScript. The website is fully responsive website which best showcases the Front End Web Developer course at Finglas Training Centre. The website is designed and developed to be fully responsive and mobile-first
