Café Website - Case Study

view Code on github

Project Background

This website was designed and developed as part of a group project as part of the traineeship I am currently doing. Cafe Bean is a fictional Café located in Finglas, Dublin. The goal of this website is to bring attention to a new Café in the Finglas Area and ultimately bring great tasting and sustainable coffee and delicious food to The local people in Finglas

Planning & Design

During the research phase of this project plan, we researched other small Irish cafés for inspiration. We picked the name for this website because we liked how it related to coffee beans and we agreed that the name Cafe Bean was an appropriate play on words. Our colour scheme was representative of the colours of coffee beans. It was important for us to choose colours which created a sense of warmth.

Development Process

The website was developed using the latest versions of HTML5, CSS and JavaScript and be to be viewed in mobile, tablet and desktop
